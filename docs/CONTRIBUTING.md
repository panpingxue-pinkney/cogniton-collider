We welcome contributions across various disciplines: Cognitive Physics, Distributed Systems, and Topological Data Analysis (TDA).

## 1. How to Submit Code

1.  Fork the repository.
2.  Create your feature branch (`git checkout -b feature/AmazingFeature`).
3.  Commit your changes (`git commit -m 'feat: Add AmazingFeature'`).
4.  Push to the branch (`git push origin feature/AmazingFeature`).
5.  Open a Pull Request (PR) against the main branch.

## 2. Academic Contributions

Academic partners should follow the guidelines outlined in the **README.md**'s Global Expert Invitation section.
